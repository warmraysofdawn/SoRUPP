# SoRUPP
Russian-based web-app which makes the sites for content authors
_It is only Russian language version. You can find English version [in special repository](https://github.com/warmraysofdawn/SoRUPP_eng)._

[![SoRUPP](https://i.postimg.cc/dQRSyy93/So-RUPP-logo.png)](https://postimg.cc/mhDwxhJW)

# Создавайте полноценные сайты-портфолио без навыков программирования!

## SoRUPP (System of Reserving Unique Personal Products, или же Система Резервирования Контента и Цифрового Наследия (СРКЦН)) - не просто программа, это:
* инструмент для создания сайтов с 12 шаблонами дизайна
* ваш шанс заявить о своём творчестве через свой личный полноценный сайт с SEO и адаптивным дизайном
* интуитивно понятный интерфейс
* дерево вашего творчества
* автоматическое создание __redirects и .htaccess
* множество других возможностей!

## 🚀 Быстрый старт

### Для Windows:
1. Установите [Python](https://python.org/downloads) версии 3.6 или старше.
2. Скачайте архив последней версии с припиской WIN из [Releases](https://github.com/warmraysofdawn/SoRUPP/releases "Перейти на страницу скачивания")
3. Распакуйте архив в пустую папку. <span style="color:red">Важно: НЕ ТЕРЯЙТЕ ФАЙЛЫ!</span>
4. Запустите `start.bat` и наслаждайтесь! Результат будет в папке "Загрузки"

### Linux && MacOS:
1. Установите [Python](https://python.org/downloads) версии 3.6 или старше.
2. Скачайте архив последней версии с припиской Linux из [Releases](https://github.com/warmraysofdawn/SoRUPP/releases "Перейти на страницу скачивания")
3. Распакуйте архив в пустую папку. <span style="color:red">Важно: НЕ ТЕРЯЙТЕ ФАЙЛЫ!</span>
4. Запустите `start.sh` и наслаждайтесь! Результат будет в папке "Загрузки"

### Для работы с чистым кодом (универсально):
1. Скачайте весь SDK следующей командой:
   ```bash
   git clone https://github.com/warmraysofdawn/SoRUPP
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите launch.py:
   ```bash
   python launch.py
   ```
   Теперь вы вещаете в вашей локальной сети GUI СРКЦН. Окно должно появиться автоматически. Если этого не произошло, попробуйте перейти по ссылке:
   http://127.0.0.1:5000

## 🌟 Основные возможности

### 📁 Управление работами
- Добавляйте свои работы (программы, литературные произведения, научные статьи, цифровое искусство и т.д.)
- Каждая работа может содержать: название, описание, дату создания, сферу/жанр/тему, теги, прикреплённые файлы (основной контент, обложка, README)
- Автоматическое хеширование файлов для контроля целостности
- Связывание работ между собой (граф зависимостей)

### 🎨 Портфолио и профиль
- Загрузите своё фото, укажите имя, цитату, биографию
- Настройте акцентный цвет сайта
- Выберите один из готовых шаблонов оформления
- Заполните SEO-метатеги для продвижения в поисковых системах

### 🖼️ Галереи изображений
- Создавайте тематические галереи для визуальных работ
- Поддерживаются форматы PNG, JPG, GIF, WebP, SVG
- Возможность редактировать галерею (добавлять/удалять изображения)

### 🔄 Резервное копирование
- Автоматические бэкапы при каждом изменении данных
- Ручное создание бэкапов с комментариями
- Просмотр списка бэкапов и выборочное восстановление файлов
- Ограничение количества хранимых бэкапов (по умолчанию 10)

### 🛡️ Проверка целостности
- Система контроля целостности файлов на основе SHA-256
- При запуске приложение проверяет соответствие хешей и сообщает о проблемах
- Восстановление повреждённых файлов из бэкапа

### 🔗 Интеграция с Git (опционально)
- Привяжите GitHub или GitLab репозиторий
- Сохраняйте свои данные в облаке под версионным контролем
- Просматривайте историю коммитов, восстанавливайте отдельные файлы из прошлых версий
- Выполняйте push и pull прямо из интерфейса

### 📤 Экспорт статического сайта
- Генерируйте полностью готовый к публикации статический сайт
- Все страницы создаются с правильными относительными ссылками
- Автоматически создаются `.htaccess` и `_redirects` для поддержки красивых URL
- Готовый ZIP-архив сохраняется в папку `Загрузки/SoRUPP_Exports`

### 📊 Экспорт данных в JSON/CSV
- Выгружайте список работ в формате JSON или CSV для внешней обработки
- Можно выбрать только нужные поля

## 🖥️ Интерфейс программы

SoRUPP поставляется с графическим интерфейсом на основе `pywebview`, который запускает локальный Flask-сервер и открывает окно браузера с панелью управления.

### Основные разделы интерфейса:
- **Главная** – краткая статистика и последние действия
- **Работы** – таблица всех работ, фильтрация по сферам/жанрам/тегам, добавление/редактирование/удаление
- **Портфолио** – настройка личных данных и шаблона
- **Галереи** – управление галереями изображений
- **Резервные копии** – просмотр, создание, восстановление
- **Целостность** – проверка и восстановление файлов
- **Git** – настройка и работа с репозиториями
- **Экспорт** – генерация сайта и выгрузка данных

## 🧰 Технические детали

### Стек технологий
- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript (vanilla)
- **GUI**: pywebview (встроенный браузер)
- **Шифрование**: cryptography (Fernet, PBKDF2)
- **Git**: PyGithub, python-gitlab
- **Разметка**: markdown

### Структура проекта
```
SoRUPP/
├── app.py                 # Основное Flask-приложение
├── launch.py              # Запуск GUI через pywebview
├── requirements.txt       # Зависимости Python
├── start.bat / start.sh   # Скрипты для быстрого запуска
├── static/
│   ├── templates/         # Шаблоны дизайна сайта
│   ├── uploads/           # Загруженные файлы пользователя
│   └── (JS, CSS файлы)
├── data/                  # JSON-файлы с данными
├── backups/               # Папка с резервными копиями
└── README.md              # Этот файл
```

### Безопасность
- Все конфиденциальные данные (токены Git) шифруются с помощью Fernet и мастер-пароля
- Мастер-пароль не является обязательным – без него Git-функции просто недоступны
- HTTP Basic Authentication может быть включена через переменные окружения (для дополнительной защиты при удалённом доступе)
- Проверка типов загружаемых файлов, защита от directory traversal

## 🎯 Как это работает?

1. **Запуск**: `launch.py` открывает GUI в отдельном окне pywebview
2. **Управление**: Вы работаете через интуитивный интерфейс, добавляя:
   - Профиль (портрет, биография, цитата)
   - Творческие работы с метаданными
   - Галереи изображений
   - Связи между работами
3. **Экспорт**: При нажатии "Сгенерировать сайт-портфолио" система:
   - Собирает все данные в структурированный архив
   - Создает SEO-оптимизированные HTML страницы
   - Генерирует необходимые конфигурационные файлы
   - Сохраняет ZIP в папку "Загрузки/SoRUPP_Exports/"

## 🛠️ Для разработчиков

### Локальная разработка:
```bash
# Клонирование репозитория
git clone https://github.com/warmraysofdawn/SoRUPP
cd SoRUPP

# Создание виртуального окружения (опционально)
python -m venv venv
source venv/bin/activate  # Linux/MacOS
# или
venv\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt

# Запуск в режиме разработки
python app.py
```

### Структура данных:
Система использует JSON файлы для хранения данных:
- `content.json` - творческие работы с метаданными
- `portfolio.json` - настройки профиля и дизайна
- `galleries.json` - коллекции изображений

### Создание собственных шаблонов:
Добавьте папку с вашим шаблоном в `static/templates/` с файлами:
- `styles.css` - стили шаблона
- `preview.jpg` - превью шаблона (опционально)
<span style="color:red">Важно: называйте файл строго style.css; в GUI название шаблона отобразится по имени папки, в которой лежит шаблоню</span>

## ❓ Часто задаваемые вопросы

**Q: Нужны ли мне навыки программирования?**
A: Нет! Интерфейс интуитивно понятен и не требует программирования.


**Q: Можно ли изменить дизайн после экспорта?**
A: Да, вы можете редактировать CSS файлы в экспортированном сайте или заменить styles.css в вашем готовом сайте на styles.css из другой папки шаблона (или вы можете просто заново запустить прилоэение - все данные уже записаны в json, вам останется только изменить шаблон дизайна)


**Q: Поддерживаются ли пользовательские домены?**
A: Да, экспортированный сайт можно разместить на любом домене.



**Q: Как часто обновляется проект?**
A: Мы регулярно выпускаем обновления с новыми функциями и исправлениями. Вы можете стимулировать дальнейшую разработку, помечая репозиторий звёздочкой и/или делясь им с друзьями.
